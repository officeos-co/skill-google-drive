## API Coverage
- Files: list, get, create, update, delete, copy, move
- Upload/Download: upload local files, download files, export Google Workspace files
- Folders: create, list contents
- Search: query-based and full-text search
- Sharing: share files, list permissions, remove permissions
- Trash: trash, untrash, empty trash
